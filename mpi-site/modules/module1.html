MPIBegin1

3 нұсқа

Тапсырма: MPI_COMM_WORLD коммуникаторына кіретін әрбір процестің бір бүтін саны бар. Оның екі еселенген мәнін шығару. Сонымен қатар, бас процесс үшін (0 ранг) MPI_COMM_WORLD коммуникаторындағы процессер санын шығару.

Тапсырма шарты түсініктемесі: Әр процесте бүтін санды оқып, оның екі еселенгенін шығару. Бас процесте қосымша процессер санын көрсету.

Шешу тәсілі

Бұл тапсырманы орындау үшін:

  1. MPI ортасын инициализациялау (MPI_Init)
  2. Әр процестің рангін алу (MPI_Comm_rank)
  3. Процессер санын алу (MPI_Comm_size)
  4. Әр процесте санды енгізу (бұл мысалда тікелей берілген)
  5. Екі еселенген мәнін шығару
  6. Бас процесс үшін (ранг 0) процессер санын шығару
  7. MPI ортасын жабу (MPI_Finalize)
mpi_begin1.c - ДҰРЫС ШЕШІМ 
#include "mpi.h"
#include <stdio.h>

int main(int argc, char *argv[]) {
    int rank, size, number;
    
    // MPI бастау
    MPI_Init(&argc, &argv);
    
    // Ранг пен өлшемді алу
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    
    // Әр процесте санды алу (бұл мысалда тікелей берілген)
    // Нақты бағдарламада scanf арқылы енгізу керек
    if (rank == 0) number = 7;
    else if (rank == 1) number = 4;
    else if (rank == 2) number = 11;
    else if (rank == 3) number = 3;
    
    // Барлық процесстер үшін есептеу және шығару
    int doubled = number * 2;
    printf("Процесс %d: сан = %d, 2 еселенгені = %d\n", 
           rank, number, doubled);
    
    // Бас процесте процессер санын көрсету
    if (rank == 0) {
        printf("================================\n");
        printf("Бас процесс (ранг 0) хабарлайды:\n");
        printf("MPI_COMM_WORLD коммуникаторында %d процесс бар\n", size);
        printf("================================\n");
    }
    
    // MPI аяқтау
    MPI_Finalize();
    
    return 0;
}

MPIBegin2

2 нұсқа

Тапсырма: Бас процесте нақты санды енгізіп, оның қарама-қарсы мәнін шығару, ал қалған процесстерде (бағынушы процесстер, рангі 0-ден үлкен) өз рангтерін шығару.

Тапсырма шарты түсініктемесі: Тек бас процесте нақты сан енгізіледі, қалғандары тек өз рангтерін көрсетеді.

Шешу тәсілі

Бұл тапсырманы орындау үшін:

  1. Бас процесте (ранг 0) нақты сан енгізу (немесе тікелей беру)
  2. Оның қарама-қарсы мәнін есептеу және шығару
  3. Бас емес процесстер үшін рангті шығару
  4. Әр процестің өзінше жұмыс істеуін қамтамасыз ету (if-else операторы)
mpi_begin2.c - ДҰРЫС ШЕШІМ 
#include "mpi.h"
#include <stdio.h>

int main(int argc, char *argv[]) {
    int rank;
    double number;
    
    // MPI бастау
    MPI_Init(&argc, &argv);
    
    // Процесс рангін алу
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    // Рангке байланысты әр түрлі әрекеттер
    if (rank == 0) {
        // Бас процесте дерек енгізу
        // Нақты бағдарламада: scanf("%lf", &number);
        number = 12.5; // Мысал мәні
        
        printf("==============================\n");
        printf("Бас процесс (ранг 0) хабарлайды:\n");
        printf("Енгізілген сан: %.2f\n", number);
        printf("Қарама-қарсы мән: %.2f\n", -number);
        printf("==============================\n");
    } else {
        // Бас емес процесстер үшін
        printf("==============================\n");
        printf("Процесс %d хабарлайды:\n", rank);
        printf("Менің рангім: %d\n", rank);
        printf("==============================\n");
    }
    
    // MPI аяқтау
    MPI_Finalize();
    
    return 0;
}
