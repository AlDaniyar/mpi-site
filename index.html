MPI - Message Passing Interface

Жоғары өнімді есептеулердің негізгі стандарты

MPI (Message Passing Interface) - бұл параллельдік бағдарламалау үшін кең таралған стандарт, процесстер арасында тиімді хабар алмасуды қамтамасыз етеді.

Бастау Практика 
#include "mpi.h"
int main(int argc, char *argv[]) {
    MPI_Init(&argc, &argv);
    int rank, size;
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    printf("Process %d of %d\n", rank, size);
    MPI_Finalize();
    return 0;
}

MPI жылдам бастау

Төмендегі мысал кодымен MPI бағдарламасын жазуға бірден кірісе аласыз:

mpi_hello.c 
#include "mpi.h"
#include <stdio.h>

int main(int argc, char *argv[]) {
    int rank, size;
    
    // MPI бастау
    MPI_Init(&argc, &argv);
    
    // Ағымдағы процестің рангін алу
    MPI_Comm_rank(MPI_COMM_WORLD, &rank);
    
    // Барлық процессер санын алу
    MPI_Comm_size(MPI_COMM_WORLD, &size);
    
    printf("Сәлем! Мен %d процесім, барлығы %d процесс\n", rank, size);
    
    // MPI аяқтау
    MPI_Finalize();
    
    return 0;
}
